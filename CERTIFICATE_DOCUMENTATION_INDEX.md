## üéì Key Concepts

### Certificate ID Format
- Format: `SZS_CERT_YYYY_MMM_NNN`
- Example: `SZS_CERT_2025_NOV_001`
- Serial resets monthly

### Workflow
1. Admin clicks "üìú Certificate" button
2. Backend generates unique certificate
3. PDF displays in modal
4. Admin can download or view

### Key Files
- Template: `SWIZ CERTI/certificate-generator/certificate/certificate_template.pdf`
- Generated: `SWIZ CERTI/certificate-generator/generated/`
- Tracking: `SWIZ CERTI/certificate-generator/serial.json`

### API Endpoints
- `POST /admin/api/generate-certificate/<candidate_id>`
- `GET /admin/api/download-certificate/<certificate_id>`
