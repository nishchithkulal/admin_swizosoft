## What You'll See

### Certificate Details
- **Candidate Name:** Displayed in large, italic font on the certificate
- **Name Format:** Automatically converted to UPPERCASE
- **Certificate ID:** Format is `SZS_CERT_YYYY_MMM_NNN`
  - YYYY = Year (e.g., 2025)
  - MMM = Month abbreviated (JAN, FEB, MAR, etc.)
  - NNN = Serial number (001, 002, 003, etc.)

### Example Certificate IDs
- First certificate in November 2025: `SZS_CERT_2025_NOV_001`
- Second certificate in November 2025: `SZS_CERT_2025_NOV_002`
- First certificate in December 2025: `SZS_CERT_2025_DEC_001` (resets monthly)

## Troubleshooting

### Certificate Won't Generate
**Issue:** Modal shows "Generating certificate..." but nothing appears

**Solution:**
1. Check internet connection
2. Try clicking the button again
3. Refresh the page and try again
4. Contact IT support if problem persists

### Name Not Visible on Certificate
**Issue:** Certificate generates but candidate name isn't shown

**Solution:**
1. Verify candidate profile has a name entered
2. Try generating again
3. Contact support if issue continues

### Download Not Working
**Issue:** Download button doesn't download the PDF

**Solution:**
1. Check browser download settings (may be blocked)
2. Try a different browser
3. Contact IT support

### Certificate Template Missing Error
**Issue:** Error message says "Certificate template not found"

**Solution:**
- This is a server-side issue
- Contact IT/technical team immediately
