## ‚ö° Quick Reference

### Files Involved
```
Frontend:  templates/admin_job_description.html (updated)
Backend:   admin_app.py - route /admin/job-description (working)
Database:  job_description table (auto-created)
```

### Key Operations
```
Add:    POST /admin/job-description with action=add
Edit:   POST /admin/job-description with action=save
Delete: POST /admin/job-description with action=delete
View:   GET /admin/job-description
```

### Database Tables
```
Main:     job_description (id, domain, description)
Synced:   approved_candidates (job_description column)
```
